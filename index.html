שלב 1: מציאת ערכים עצמיים
נמצא את הערכים העצמיים של המטריצה A על ידי פתרון |λI − A| = 0

שאלה מנחה

חשבו את הפולינום האופייני:

|λI − A| = λ² − 2λ + 5 = 0

רמז: השתמשו בנוסחה הריבועית

λ = ± i

הזינו מספרים: למשל α=1, β=2

פתרון שלב 1

הפולינום האופייני:
|λI − A| = (λ−3)(λ+1) − (−2)(4) = λ² − 2λ + 5
פתרון המשוואה הריבועית:
λ = (2 ± √(4−20))/2 = (2 ± √(−16))/2 = (2 ± 4i)/2

הערכים העצמיים

λ = 1 ± 2i
שלב 2: מציאת וקטור עצמי
נמצא וקטור עצמי v עבור λ = 1 + 2i

שאלה מנחה

פתרו את המערכת (λI − A)v = 0:

(−2+2i)v₁ + 2v₂ = 0

רמז: בחרו v₁ = 1 ומצאו את v₂

v = [1, + i]T

הזינו את הרכיב השני: למשל 1 ו־ -1 עבור 1−i

פתרון שלב 2

המטריצה (λI − A) עבור λ = 1+2i:
[(−2+2i), 2; −4, (2+2i)]
מהמשוואה הראשונה:
(−2+2i)v₁ + 2v₂ = 0 → v₂ = (1−i)v₁
בחירת v₁ = 1:
v₂ = 1 − i

הוקטור העצמי

v = [1, 1−i]T
שלב 3: הפתרון הכללי
נפרק את הפתרון המרוכב לחלקים ממשי ומדומה

שאלה מנחה

הפתרון הכללי הוא:

x(t) = C₁et[cos2t; cos2t+sin2t] + C₂et[sin2t; sin2t−cos2t]

זכרו: eαt(cosβt + i·sinβt)

y₂(t) = et( ·sin2t − ·cos2t)

הזינו מספרים: 1 או -1

🧐 Digital Friend - שלב 3

פתרון שלב 3

הפתרון המרוכב:
e(1+2i)tv = ete2it[1; 1−i]
פירוק באמצעות נוסחת אוילר eiθ = cosθ + i·sinθ:
= et(cos2t + i·sin2t)[1; 1−i]
פתיחת המכפלה - נכפיל כל רכיב בוקטור:
רכיב ראשון: (cos2t + i·sin2t)·1 = cos2t + i·sin2t

רכיב שני: (cos2t + i·sin2t)·(1−i)
  = cos2t·1 + cos2t·(−i) + i·sin2t·1 + i·sin2t·(−i)
  = cos2t − i·cos2t + i·sin2t − i²·sin2t
  = cos2t − i·cos2t + i·sin2t + sin2t   (כי i² = −1)
  = (cos2t + sin2t) + i·(sin2t − cos2t)
הפתרון המרוכב המלא:
et[cos2t + i·sin2t; (cos2t+sin2t) + i·(sin2t−cos2t)]
החלק הממשי (פתרון ראשון x₁):
x₁(t) = et[cos2t; cos2t + sin2t]
החלק המדומה (פתרון שני x₂):
x₂(t) = et[sin2t; sin2t − cos2t]

הפתרון הכללי

x(t) = C₁et[cos2t; cos2t+sin2t] + C₂et[sin2t; sin2t−cos2t]

פתרון שלב 4

הצבת t = 0:
[1; 0] = C₁[cos0; cos0+sin0] + C₂[sin0; sin0−cos0]
פישוט:
[1; 0] = C₁[1; 1] + C₂[0; −1] = [C₁; C₁ − C₂]
מהמשוואה הראשונה: C₁ = 1
מהמשוואה השנייה: C₁ − C₂ = 0 → C₂ = 1

הקבועים

C₁ = 1, C₂ = 1
